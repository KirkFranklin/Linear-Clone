# ◇ Linear Clone — Modern Web App & Issue Tracker

A high-performance, dark-mode landing page and interactive Kanban issue tracker inspired by Linear. Built with React, Vite, React Router, CSS glassmorphism, and localStorage state persistence.

 **Live Demo:** [Your Vercel App Link Here]https://linear-clone-gjf6.vercel.app/

---

##  Features

- **Dynamic Workflows Preview:** Tab switcher showcasing issue management, cycles, and product roadmaps.
- **Interactive Pricing Table:** Toggle seamlessly between monthly and annual billing with live price recalculations.
- **Accordion FAQ:** Expandable questions and answers built with clean React state hooks.
- **Responsive Navigation:** Floating glassmorphic desktop bar transitioning to a smooth mobile drawer menu.
- **Full-Featured Kanban Board (`/app`):**
  - Create, Read, Update, and Delete (CRUD) issues.
  - Tag tasks by priority (**Urgent**, **High**, **Medium**, **Low**).
  - Move issues between status columns (**Todo**, **In Progress**, **Done**).
  - Persistence via `localStorage`.

---

## Tech Stack

- **Frontend:** React 18, Vite
- **Routing:** React Router v6
- **Styling:** Modern CSS (CSS Variables, Flexbox/Grid, Glassmorphism)
- **Deployment:** Vercel
